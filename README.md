# en-scan

Usage: en-scan DAYS

 DAYS number of days in the past you want to scan.
      You can use 0 to scan just today.

To run from remote:

 curl -sk https://raw.githubusercontent.com/purplesrl/en-scan/master/en-scan | python /dev/stdin [DAYS]
